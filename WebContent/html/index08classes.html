Using The class Attribute

  1. The class attribute specifies one or more class names for an HTML element.
  2. The class name can be used by CSS and JavaScript to perform certain tasks for elements with the specified class name.
  3. In CSS, to select elements with a specific class, write a period (.) character, followed by the name of the class:

London

London is the capital of England.

Paris

Paris is the capital of France.

Tokyo

Tokyo is the capital of Japan.
